💳 PayPal Checkout — React + Vite

A minimalist PayPal Smart Checkout integration using modern React, Vite, and Tailwind CSS.

🔗 Live Demo

🚀 Features

  • ⚛️ Built with React + Vite
  • 🎨 Styled using Tailwind CSS
  • 💳 Integrated PayPal Smart Payment Buttons
  • 🔄 Routing with React Router (Home, Checkout, Success, Cancel)
  • 📱 Mobile-responsive and clean design
  • ☁️ Hosted on Vercel

🧩 Tech Stack

React, Vite, Tailwind CSS, React Router DOM, @paypal/react-paypal-js

📦 Installation & Setup

git clone https://github.com/anik588/paypal-payment-project.git
cd paypal-payment-project
npm install
npm run dev

🛠️ Build for Production

npm run build

Then deploy the /dist folder to any static host. (Vercel does this automatically.)

🧪 Live Test

  • Visit the Live App
  • Go to /checkout and test the PayPal flow
  • Redirects to /success or /cancel
